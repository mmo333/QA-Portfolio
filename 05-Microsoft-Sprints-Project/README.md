# Microsoft & Sprints QA Simulation Project

## 📌 Overview
This project was part of Microsoft & Sprints Summer Camp training program.

It involved testing a demo e-commerce application (DemoBlaze) based on provided requirements.

---

## 🧩 Activities Performed

- Requirement analysis
- Writing test cases based on requirements
- Functional testing of e-commerce features
- Identifying and reporting bugs
- Regression testing after fixes

---

## 🐞 Bug Reporting

Defects were documented with:
- Steps to reproduce
- Expected vs actual results
- Severity & priority classification

---

## 🔁 Workflow

Simulated QA lifecycle:
- Open
- In Progress
- Retest
- Closed

---

## 🎯 Outcome

This project demonstrates real-world QA simulation experience in:
- Requirement-based testing
- E-commerce testing scenarios
- Defect lifecycle management
